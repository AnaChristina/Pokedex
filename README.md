## Executando Projeto

Para executar o projeto, primeiro faça um clone do repositório utilizando terminal, atente-se ao local onde irá salvar. <br>
`git clone https://github.com/AnaChristina/Pokedex.git`

Navegue até o diretóriodo em que você salvou o projeto<br>
`cd pokedex`

Abra o projeto de preferência no Visual Studio Code, abra um novo terminal no Visual Studio e digite os seguintes comandos:<br>
`npm install`<br>
`npm start`<br>
(certifique-se de estar dentro da pasta do projeto.)
O projeto será aberto em: `http://localhost:3000/` se o projeto não abrir automaticamente, é so acessar `http://localhost:3000/` no navegador.

## Uso
- Ao abrir a página, você será apresentado a uma lista de Pokémons à esquerda, carregada a partir da API.
- Clique no nome de um Pokémon dentro da lista para visualizar a imagem e as informações detalhadas sobre ele.
- Utilize a barra de pesquisa para encontrar um Pokémon específico.
- Para alterar tema, clique no botão vermelho abaixo da tela de informações do pokemon.
